遇到的问题：

Q1：为什么每次hexo d后我的域名都不起作用了？

A1：在source目录里创建CNAME,内容为你的域名。

Q2：为什么每次hexo d后出现ERROR Deployer not found:

A2：执行下列语句。

$ npm install --save hexo-deployer-git
